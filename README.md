## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- node
  ```sh
  node -v
  ```
- yarn
  ```sh
  npm install -g yarn
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/BrunoViveiros/vitae.git
   ```
2. Go to the vitae directory
   ```sh
   cd vitae
   ```
3. Install Yarn packages
   ```sh
    yarn install
   ```
